# Todo-List
Todo-List를 이용해 일상에서의 자신이 해야할 일과, 생활을 효율적으로 관리할 수 있다.

# Skill

# 구현해야할 기능들(해야할 것들)
- 퍼블리싱
- 자신이 등록한 리스트 중에서 완료한 것들은 완료했다고 할 수 있다.

==== 퍼블리싱 이후 ====
- JSON SERVER에서 리스트를 불러올 수 있다.
- 자신이 해야할 일들을 등록할 수 있다 그리고 JSON SERVER에 저장할 수 있다.
- 자신이 해야할 일들을 삭제할 수 있다.
